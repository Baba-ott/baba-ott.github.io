Welcome

Starting from scratch (literally) - My first experience with any type of programming came when I was 11 with the software application Scratch. Scratch is a slightly GUI friendly programm that lets users build games with command lines designed for a younger audience. It was in the beginning of middle school that I had to use algorthmic thinking in designing and creating a game. It was my most enjoyable activity to learn about in school at the time. Ever since then, my interest in technology and the creation behind it has only grown!

I love ICT because it is something that has grown up with me. As time has passed I've truly felt connected with the use of technology, specifically anything software and hardware related. As you can see above, is a blackberry tablet, which was the first device I started using besides a basic nokia flip phone.
